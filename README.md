# Console-Snake
## Description
Console-Snake is a console snake game with basic customization options (colors, speed, etc.)
## Installation
Clone Repository:
```powershell
git clone https://github.com/W1nsh/Console-Snake
```
Install requirements:
```powershell
pip install -r requirements.txt
```
## Start
Go to the directory with console snake
```powershell
cd console-snake
```
You can start game:
```powershell
python main.py
```
## Functions
- Customizate game for you: colors, speeds, time life of food and other.
- Playing wihtout internet: if internet is breaking you will playing to the best console snake whatever
